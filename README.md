Running simply_fit
1) Install Docker
2) Clone simply_fit
3) Run "docker build -t yourname/simply_fit /location/of/cloned/repo/"
4) Run "docker run --name=hello_django \
    --detach=true \
    --restart=always \
    --publish=8001:8000 \
    michal/hello_django:latest"
5) Access localhost:8001 from favorite browser
